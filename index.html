📤 Export Menu Data

Edit this JSON directly or paste in a saved JSON, then click Apply / Import JSON to load it into the live menu. To make changes permanent across all devices, download and replace items.json in your project.

📦 Backup Snapshot

A backup is automatically created from items.json on first load. Downloaded backups are named with a timestamp and belong in the backups/ folder in your project. You can load any saved backup file back in at any time.

ℹ️ How It Works

Live Edits: Your changes save to browser storage instantly.
TV Displays: They check for live edits first, then fall back to items.json.
Making It Permanent: Export your changes and save to items.json, then push to GitHub.
Reset: Wipes all live edits and reloads from items.json.
